# Utils

Repository-level utility scripts.

| File | Purpose |
|------|---------|
| `pushscripts.sh` | Deploy analysis scripts to multiple project directories |
| `zip.sh` | Create versioned archives of analysis results |
| `ntslim_converter.ipynb` | Jupyter notebook for NAMD/AMBER trajectory format conversion |

For trajectory analysis utilities (makePDB, strip, cpptraj_process), see `md_protocol/utils/`.
For the full cpptraj analysis pipeline, see `md_protocol/03_analysis/claude_cpptraj_pipeline/`.
For the WESTPA bin visualizer, see `westpa/tools/binviz/`.
